# Help is coming here

This is the help content for the Group Practice feature.

## Getting Started

Use this section to practice reciting Bhagavad Gita verses with a group.

## Features

- **Chapter Selection**: Choose which chapter to practice
- **Participants**: Set the number of participants in your practice session
- **Your Turn**: Specify which turn is yours
- **Wait Mode**: Choose how to wait between turns

## Controls

- **Play**: Start the practice session
- **Pause**: Pause the current playback
- **Stop**: Stop the practice session

For more information, please refer to the main documentation.
