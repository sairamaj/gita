Use this section to practice reciting Bhagavad Gita verses with a group. 

## How It Works

1. **Setup**: Configure the number of participants and your turn number
2. **Practice Cycle**: The program plays shlokas for each participant in order
3. **Your Turn**: When it's your turn, you recite your shloka
4. **Other Participants**: The program automatically plays shlokas for other participants

## Features

- **Chapter Selection**: The currently selected chapter is displayed
- **Participants**: Set the total number of participants in your practice session
- **Your Turn**: Specify which turn number is yours (1, 2, 3, etc.)
- **Wait Mode**: Choose how to wait between turns:
  - **Keyboard Hit**: The program waits for you to press OK to finish your turn and proceed
  - **Duration**: The program waits for a specified number of seconds before continuing
- **Duration**: When using Duration wait mode, specify how many seconds to wait (this is your time to recite)
- **Repeat Your Sloka**: Check this box if you want to hear your sloka again after you recite it
- **Playback Speed**: Adjust the playback speed from 0.5x to 2.0x using the slider
- **Number of Stanzas**: Set how many stanzas each other participant will recite

## Controls

- **Play**: Start the group practice session
- **Stop**: Stop the practice session at any time

## Practice Flow

1. Select your desired settings (Number of Participants, Your Turn, Wait Mode, Duration, Playback Speed, etc.)
2. Click **Play** to start
3. The program plays shlokas for each participant in sequence
4. When it's your turn:
   - If **Keyboard Hit** mode: Click OK when you finish reciting to proceed
   - If **Duration** mode: Wait for the specified duration to pass
5. The cycle continues with the next participant
